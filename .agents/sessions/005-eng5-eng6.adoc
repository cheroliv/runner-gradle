  * Verifie `info.title` present
  * Verifie les 3 paths requis
  * Verifie les 6 schemas requis
  * Compte les paths/schemas
  * Execution : BUILD SUCCESSFUL

=== ENG-6 — Chatbot CI/CD conversationnel

  * Modele par defaut : `deepseek-v4-pro:cloud` (parametrable)
  * Integration curl vers Ollama API `/api/chat`
  * Parsing JSON avec `jq`
  * Chmod +x applique
